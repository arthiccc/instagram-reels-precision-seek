## ig-reels-native-seek:
non-intrusive seek bar for instagram web.

## features
- zero-bloat ui.
- hotkey support.
- optimized for legacy hardware (low cpu/ram footprint).

## usage
- seek bar at bottom of reels.
- shift + arrow keys to seek ±5s.
- click/drag on seek bar to jump.
