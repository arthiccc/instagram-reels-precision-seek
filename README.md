## ig-reels-native-seek:
non-intrusive seek bar for instagram web.

## features
- zero-bloat ui.
- hotkey support (shift+arrows, space to pause).
- duration display next to save button.
- seeking pauses video for smooth scrub.
- optimized for legacy hardware (low cpu/ram footprint).

## usage
- seek bar at bottom of reels.
- shift + arrow keys to seek ±5s.
- space bar to pause/play.
- click/drag on seek bar to jump.
- duration shown next to save/bookmark button.
